Best platform features

Carrier is an open-source performance tests orchestration and execution platform designed to run big-scale load and performance tests from your pipeline utilizing the power and elasticity of the Cloud or K8S clusters.

Real-time Reporting

Carrier provides accurate real-time reporting of performance test results. Grouped errors logs in one table by request name/method. Including resource utilizations on LGs

Cloud Integration

Carrier is designed to run big-scale load and performance tests from your pipeline utilizing the power and elasticity of the Cloud or K8S clusters.

Configurable Thresholds

Carrier provides the ability to configure thresholds on response time / error rate / throughput for any request.

Performance Comparison

Carrier allows performance results to be automatically compared versus SLOs and KPIs collected from previous test runs.

Open-Source Tools Integration

Performance tests developed using leading open-source tools like JMeter, Gatling, Lighthouse, sitespeed.io can be easily run, analyzed, and compared using the platform.

On-Premises Installation

Carrier instance can be deployed on-premise behind your firewall or in the Cloud. All your data is located on your own instance, and it is 100% secured.

Performance Testing Services

Professional Performance Testing Services for Your Business

As experts in performance testing, our team is dedicated to helping achieve your goals, while maximizing efficiency and minimizing risks. We offer tailored performance testing and consulting services to meet every client’s need. Our team of experienced performance analysts support product teams in developing scalability and resilience strategy, as well as aid developers in fine-tuning and bottlenecks detection.

Backend Performance Testing

Backend performance testing with the Carrier provides an ability to execute any type of tests (capacity, longevity, stable response time tests, etc.) using open source tools like JMeter and Gatling. Test results could be compared between different runs or against configured thresholds.

Frontend Performance Testing

Frontend performance testing with the Carrier provides comprehensive insights into website performance and helps identify and address issues that may be impacting user experience. Utilizing the power of Sitespeed.io and user-flow based Lighthouse tools you can get comprehensive, page-specific reports, and invaluable suggestions for optimization.

Load Generator Configuration And Test Data Preparation

Carrier provides an ability to configure your load generators with custom extensions (like JMeter plugins). Also, you can split your test data between several load generators to produce a big scale load.

Scalability Performance Testing

We conduct scalability tests to define optimal environment configuration and/or scaling strategy. We work with development/DevOps teams to investigate the root causes of issues and validate the impact of changes.

Handover and Education

We ensure a smooth handover and provide education on how to run, analyze, and maintain performance test frameworks through our expert-led training sessions.

Hypercare

With our Hypercare service, the delivery team can conduct tests for new builds on their own, ensuring continuous performance monitoring. We provide subject matter expertise and help in performance testing related work.
